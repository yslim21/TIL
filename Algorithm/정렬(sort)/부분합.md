# 부분합
[부분합](https://www.acmicpc.net/problem/1806)

## 개념
+ 투포인터 이용
+ start: 부분 수열의 시작 지점 / cur: 부분 수열의 끝 지점
+ 현재지점이 N을 넘어서면 끝까지 탐색한 것이므로 while문 종료
+ start부터 cur-1까지의 합이 S이상인 경우
  + cur-start의 값 구해서 최소 길이인지 확인하고 최소 길이면 minLen값 갱신 
+ start부터 cur-1까지의 합이 S미만인 경우
  + cur++

## 주의
+ 처음에는 while문의 종료조건을 cur < N으로 했는데 sum > S이고 cur==N이지만 start를 오른쪽으로 이동시키면서 최소 길이를 구할 수 있는 경우를 놓치게됨 -> 종료조건을 while문 안쪽으로 수정해서 sum > S이고 cur == N이더라도 오른쪽으로 start 이동시키면서 S이상인 최소길이 구할 수 있게함 

## 풀이
```java
import java.util.*;
import java.io.*;

public class Main{
    public static void main(String[] args) throws IOException{
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        StringTokenizer st = new StringTokenizer(br.readLine());
        
        int N = Integer.parseInt(st.nextToken());
        int S = Integer.parseInt(st.nextToken());
        
        int[] arr = new int[N];
        st = new StringTokenizer(br.readLine());
        for (int i = 0; i < N; i++) {
            arr[i] = Integer.parseInt(st.nextToken());
        }
         
        int minLen = N+1; 
        int cur =0; 
        int start = 0; 
        int sum =0; 
        while(true){
            if(sum >= S){
                minLen = Math.min(minLen, cur - start);
                sum -= arr[start]; //합이 S 이상이므로 sum에서 start 빼면서 len길이 찾기 
                start++;
            }else{
                if(cur == N){
                    break;
                }
                sum += arr[cur];
                cur++;
            }
        }
        
        if(minLen == N+1){
            System.out.println(0);
        }else{
            System.out.println(minLen); 
        } 
           
    }
}
```
