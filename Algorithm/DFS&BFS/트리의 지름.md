# 트리의 지름

[트리의 지름](https://www.acmicpc.net/problem/1167)

## 개념
+ BFS 이용 
+ 임의의 노드에서 가장 긴 경로로 연결돼 있는 노드는 트리의 지름에 해당하는 두 노드 중 하나임 
+ 우선 BFS 한번 돌려서 임의 지점에서 가장 먼 지점을 찾음
+ 해당 지점에서 가장 먼 지점을 찾고 두 지점 사이의 거리를 구하면 그게 지름임 
  + 한 지점에서 다른 지점까지의 거리를 disatnce 배열에 계속 업데이트해가면서 구하자 

## 풀이
```java
import java.io.*;
import java.util.*;


class Main{
    static int N;
    static boolean visited[];
    static int[] distance;
    static ArrayList<Point>[] A;
    
    public static void main(String[] args)throws IOException{
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        
        StringTokenizer st = new StringTokenizer(br.readLine());
        N = Integer.parseInt(st.nextToken());
        
        A = new ArrayList[N+1];
        
        for(int i=1; i <= N; i++){
            A[i] = new ArrayList<Point>();
        }
        
        for(int i =0; i < N; i++){
            st = new StringTokenizer(br.readLine());
            int s = Integer.parseInt(st.nextToken());
            while(true){
                int e = Integer.parseInt(st.nextToken());
                if(e==-1){
                    break;
                }
                int v = Integer.parseInt(st.nextToken());
                A[s].add(new Point(e,v));
            }
        }
        
        distance = new int[N+1];
        visited = new boolean[N+1];
        
        BFS(1);
        int Max =1;
        
        for(int i =2; i <=N; i++){
            if(distance[Max] < distance[i]){
                Max =i;
            }
        }
        
        distance = new int[N+1];
        visited = new boolean[N+1];
        BFS(Max);
        Arrays.sort(distance);
        System.out.println(distance[N]);
        
        
    }
    
    static void BFS(int x){
        Queue<Point> queue = new LinkedList<Point>();
        queue.add(new Point(x,0));
        
        visited[x] = true;
        
        while(!queue.isEmpty()){
            Point start = queue.poll();
            
            for(Point i:A[start.e]){
                int nexte = i.e;
                int nextvalue = i.value;
                
                if(!visited[nexte]){
                    visited[nexte] = true;
                    queue.add(new Point(nexte,nextvalue));
                    distance[nexte] = distance[start.e] + nextvalue; 
                }
            }
        }
    }
    
    static class Point{
        int e;
        int value;
        
        public Point(int e, int value){
            this.e = e;
            this.value = value; 
        }
    }
}
```
