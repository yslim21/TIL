# 숫자고르기

[숫자고르기](https://www.acmicpc.net/problem/2668)

## 개념
+ DFS 이용
+ 윗 줄의 수를 출발지로 아랫줄의 수를 도착지로 그래프 그려보기 
+ 1->3, 3->1 / 1->3, 3->2, 2->1 이런식으로 사이클이 연결되면 정답 리스트에 추가 
+ (1,1)~(N,N)까지 반복문 돌면서 사이클이 존재하는지 각각 DFS로 탐색 

## 풀이
```java
import java.io.*;
import java.util.*;

public class Main{
    static int N;
    static int[] arr;
    static boolean[] visited;
    static ArrayList<Integer> ans = new ArrayList<>();
    
    public static void main(String[] args) throws Exception{
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        
        N = Integer.parseInt(br.readLine());
        arr = new int[N+1];
        
        for(int i =1; i <=N; i++){
            arr[i] = Integer.parseInt(br.readLine());
        }
        
        for(int i =1; i <=N; i++){
            visited= new boolean[N+1];
            dfs(i,i);
        }
        
        Collections.sort(ans);
        
        System.out.println(ans.size());
        for(int x: ans){
            System.out.println(x);
        }
    }
    
    static void dfs(int start, int end){
        visited[end] = true; // start가 아니라 end로 해야함
        int next = arr[end]; 
        
        if(!visited[next]){
            dfs(start, next);
        }else{
            if( next == start){
                ans.add(start);
            }
        }
    }
}
```
