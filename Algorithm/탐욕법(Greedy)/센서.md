# 센서

[센서](https://www.acmicpc.net/problem/2212)

## 개념
+ 1) 센서의 개수 <= 집중국의 개수; 집중국 하나당 센서 1개를 담당하면 되니까 최소거리가 0임 
+ 2) 센서의 개수 > 집중국의 개수; 집중국 K개로 N개의 센서를 커버할 때 전체 거리의 합을 최소로 하려며 "가장 큰 간격"을 끊어야함 
  + diff배열에 각 센서 간 간격 담은 후 오름차순 정렬
  + 반복문으로
    + N-K인덱스 전까지 돌면서 answer에 더함 
    + 인덱스가 높아질수록 간격이 크다는 의미이므로 간격이 큰 센서들은 기지국 하나가 담당하는게 최소거리임 
+ 예제1을 예시로 보면 6-2해서 최소 4개의 간격을 줄여야하는데 그래서 센서를 묶는다면 간격이 적은 센서들을 묶는 것임 

```java
import java.util.*;
import java.io.*;

public class Main {
 
    public static void main(String[] args) throws Exception {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        StringTokenizer st;
 
        int N = Integer.parseInt(br.readLine());
        int K = Integer.parseInt(br.readLine());
 
        if (K >= N) {
            System.out.println(0);
            return;
        }
 
        int[] censor = new int[N];
        st = new StringTokenizer(br.readLine());
        for (int i = 0; i < N; i++) {
            censor[i] = Integer.parseInt(st.nextToken());
        }
        Arrays.sort(censor);
 
        int[] dif = new int[N - 1];
        for (int i = 0; i < N - 1; i++) {
            dif[i] = censor[i + 1] - censor[i];
        }
        Arrays.sort(dif);
 
        int answer = 0;
        for (int i = 0; i < N - K; i++) {
            answer += dif[i];
        }
 
        System.out.println(answer);
    }
 
}

```
