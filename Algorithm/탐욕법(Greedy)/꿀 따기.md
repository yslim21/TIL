# 꿀 따기

[꿀 따기](https://www.acmicpc.net/problem/21758)

## 개념
+ 그리디 + 누적합 이용 
+ 3가지의 경우가 있음
  + 벌 - 벌 - 꿀 : 벌1과 꿀을 양끝에 고정시키고 벌2를 자유롭게 배치한 후 최대값 찾기
  + 꿀 - 벌 - 벌 : 꿀과 벌1을 양끝에 고정시키고 벌2를 자유롭게 배치한 후 최대값 찾기 
  + 벌 - 꿀 - 벌 : 벌1, 벌2를 양끝에 고정시키고 꿀을 자유롭게 배치한 후 최대값 찾기 

```java
import java.io.*; 
import java.util.*; 

public class Main{
    public static void main(String[] args) throws Exception{
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        StringTokenizer st;
        
        int N = Integer.parseInt(br.readLine());
        int[] arr = new int[N];
        
        st = new StringTokenizer(br.readLine());
        for(int i =0; i < N; i++){
            arr[i] = Integer.parseInt(st.nextToken());
        }
        
        int[] prefix = new int[N];
        int max = 0; 
        prefix[0] = arr[0];
        for(int i=1; i<N; i++){
            prefix[i] = prefix[i-1]+arr[i]; 
        }
        
        //벌 - 벌 - 꿀
        for(int i =1; i <N-1; i++){
            int sum = (prefix[N-1]-arr[0]-arr[i]) + (prefix[N-1] - prefix[i]);
            max = Math.max(max, sum);
        }
        
        //꿀 - 벌 - 벌
        for(int i=1; i < N-1; i++){
            int sum = (prefix[N-2] - arr[i]) + (prefix[i-1]);
            max = Math.max(max, sum);
        }
        
        //벌 - 꿀 - 벌
        for(int i=1; i < N-1; i++){
            int sum = (prefix[i]-arr[0]) + (prefix[N-2]-prefix[i-1]);
            max = Math.max(max,sum);
        }
        
        System.out.println(max);
    }
}
```
