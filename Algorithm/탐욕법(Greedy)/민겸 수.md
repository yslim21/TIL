# 민겸 수 

[민겸 수](https://www.acmicpc.net/problem/21314)

## 개념
+ 가장 큰 값 찾는 방법
  + M이 K를 만날때까지 상태 유지 ex) MMK -> 500, MMMK -> 5000
  + M이 연속될 시 1로 상태 변환 ex) MMM -> 111
+ 가장 작은 값 찾는 방법 
  + M이 연속될시 K가 나오기 전까지 상태 변환 ex) MM -> 10 
  + K는 바로 5로 변환

## 풀이
```java
import java.io.*;
import java.util.*;

public class Main{
    public static void main(String[] args) throws Exception{
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        
        StringTokenizer st;
        
        String line = br.readLine();
        
        String max = bigNum(line);
        String min = smallNum(line);
        
        System.out.println(max);
        System.out.println(min);  
        
    }
    
    static String bigNum(String line){
        StringBuilder sb = new StringBuilder();
        int mCnt = 0;
        for(int i =0; i < line.length(); i++){
            if(line.charAt(i) == 'K'){
                sb.append("5");
                for(int j =0; j < mCnt; j++){
                    sb.append("0");
                }
                mCnt =0;
            }else{
                mCnt++;
            }
        }
        for(int i =0; i <mCnt; i++){
            sb.append("1");
        }
        
        return sb.toString();
    }
    
    static String smallNum(String line){
        StringBuilder sb = new StringBuilder();
        int mCnt =0; 
        for(int i =0; i<line.length();i++){
            if(line.charAt(i) == 'K'){
                if(mCnt > 0){
                    sb.append("1");
                    for(int j =0; j < mCnt-1; j++){
                        sb.append("0");
                    }
                }
                mCnt =0; 
                sb.append("5");
            }else{
                mCnt++;
            }
        }
        if(mCnt >0){
            sb.append("1");
            mCnt--;
        }
        for(int i =0; i < mCnt; i++){
            sb.append("0");
        }
        
        return sb.toString();
    }
}
```
