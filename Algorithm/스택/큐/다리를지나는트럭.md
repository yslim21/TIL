# 다리를 지나는 트럭 

[다리를 지나는 트럭](https://school.programmers.co.kr/learn/courses/30/lessons/42583)

## 개념
+ 먼저 들어온 트럭이 먼저 나가므로 queue 사용 
+ time 추가하는 경우 - 트럭을 올릴때 또는 트럭이 무게 때문에 다리에 못 올라가서 1초 허비했을 때 
  + 나갈 때는 왜 타임 추가 안해 ? 
+ 트럭 리스트 반복문 돌기
  + 다리에 아무것도 없으면 트럭 올리기 -> 시간 ++ , 무게 + 
  + 다리에 트럭이 꽉찬 상태면 -> 트럭 하나 내보내기 무게 - 
  + 다리에 자리 있음
    + 무게상 트럭 올릴 수 있는 경우 -> 시간 ++, 무게 + 
    + 무게 때문에 트럭 못올라가는 경우 -> 시간 ++, 다리에 빈자리 하나 올리기 

## 풀이

```java
import java.io.*;
import java.util.*; 

class Solution {
    public int solution(int bridge_length, int weight, int[] truck_weights) {
        int answer = 0;
        //트럭은 1초에 한대씩 올라갈 수 있음(동시에 x)
        
        Queue<Integer> queue = new LinkedList<>();
        
        int sum = 0; 
        int time = 0; 
        
        for(int i =0; i < truck_weights.length; i++){
            int truck = truck_weights[i];
            
            while(true){
                //다리에 아무것도 없음 -> 큐에올리기 
                if(queue.isEmpty()){
                    queue.add(truck);
                    sum += truck; 
                    time++; 
                    break;
                }else if(queue.size() == bridge_length){//다리 꽉참
                    sum -= queue.poll();
                }else{ // 다리에 트럭은 있지만 꽉차진 않은 경우 
                    if(sum + truck <= weight){ //하나 더 올릴수 있음 
                        sum += truck;
                        queue.add(truck);
                        time++; 
                        break;
                    }else{ // 자리는 있지만 무게상 올릴 수 없음 -> 시간을 ++해서 트럭 한개 나가도록 
                        time++; 
                        queue.add(0);
                    }
                    
                }
            }
        }
        
        
        return time+bridge_length;//마지막 트럭이 다리 전체 지나가는 사간 + 트럭 올리는 시간 
    }
}
```
