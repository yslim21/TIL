# 소수&팰린드롬
[소수&팰린드롬](https://www.acmicpc.net/problem/1747)

## 개념
+ 에라토스테네스 이용
+ 2부터 반복문 돌기
  + 선택한 수의 배수는 모두 소수가 아니므로 삭제 
+ N부터 반복문 돌기
  + 팰린드롬인지 검사
  + 정수 -> char배열 형태로 변경하여 검사
    ```java
    char[] temp = String.valueOf(result).toCharArray();
    ```

## 풀이
```java
import java.util.*;
import java.io.*;

public class Main{
    public static void main(String[] args){
        Scanner in = new Scanner(System.in);
        int N = in.nextInt();
        
        int[] A = new int[10000001];
        
        for(int i=2; i <A.length; i++){
            A[i] = i; 
        }
        
        for(int i=2; i <=Math.sqrt(A.length); i++){
            if(A[i] ==0){
                continue; 
            }
            
            for(int j =i+i; j <A.length; j= j +i){
                A[j]=0;
                
            }
        }
        
        int i = N;
        while(true){
            if(A[i] !=0){
                int result = A[i];
                
                char[] temp = String.valueOf(result).toCharArray();
                int s =0;
                int e = temp.length-1;
                boolean isPalind = true; 
                
                while(s<e){
                    if(temp[s] == temp[e]){
                        s++;
                        e--;
                    }else{
                       isPalind = false;
                        break;
                    }
                }
                
                if(isPalind == true){
                    System.out.println(result);
                    break;
                }
            }
            i++;
        }
        
       
    }
}
```
