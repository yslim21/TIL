# 카펫

[카펫](https://school.programmers.co.kr/learn/courses/30/lessons/42839)

## 개념
+ 세로x가로=(brown + yellow)가 되는 pair 찾기
+ pair 중 가로*2+세로*2-2가 brown인 수가 정답

## 풀이
```java
import java.io.*;
import java.util.*;

class Solution {
    public int[] solution(int brown, int yellow) {
        int[] answer = new int[2];
        
        for(int i =2; i<= Math.sqrt(brown+yellow); i++){
            if((brown+yellow)%i == 0){
                int col = i;
                int row = (brown+yellow)/i;
                
                if((col*2 + row*2 -4) == brown){
                    answer[0] = row;
                    answer[1] = col;
                    break;
                }
            }
        } 
        
        return answer;
    }
}
```
