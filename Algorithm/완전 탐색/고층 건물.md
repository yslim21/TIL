# 고층 건물

[고층 건물](https://www.acmicpc.net/problem/1027)

## 개념
+ A건물에서 B건물로 선분을 그었을때 다른 건물들을 지나지 않으려면
  + B가 A의 왼쪽에 있는 경우: A와 B 사이에 있는 다른 건물들과 선분을 그었을 때의 기울기보다 A와 B사이의 기울기가 작아야함 
  + B가 A의 오른쪽에 있는 경우: A와 B 사이에 있는 다른 건물들과 선분을 그었을때의 기울기보다 A와 B사이의 기울기가 커야함 

## 풀이
```java
import java.io.*;
import java.util.*;

public class Main{
    public static void main(String[] args) throws Exception{
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        StringTokenizer st = new StringTokenizer(br.readLine());
                                               
        int N = Integer.parseInt(st.nextToken());
        int[] buildings = new int[N];
        
        st = new StringTokenizer(br.readLine());
        for(int i =0; i < N; i++){
            buildings[i] = Integer.parseInt(st.nextToken());
        }
        
        int ans =0; 
        
        for(int i =0; i < N; i++){
            int count =0; 
            double temp =0;
            
            //왼쪽 
            for(int j =i-1; j >=0; j-- ){
                double slope = (double)(buildings[i] - buildings[j])/(i-j);
                
                if(j == i-1 || slope < temp){
                    count++;
                    temp= slope;
                }
            }
             
            //오른쪽 
            for(int j=i+1; j<N; j++){
                double slope = (double)(buildings[j] - buildings[i])/(j-i);
                
                if(j == i+1 || slope > temp){
                    count++;
                    temp=slope;
                }
            }
            
            ans = Math.max(ans, count);
            
        }  
        
        System.out.println(ans);
    }
}
```
