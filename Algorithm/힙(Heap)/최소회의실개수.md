# 최소 회의실 개수 
[최소 회의실 개수](https://www.acmicpc.net/problem/19598)

## 개념
+ 회의 시작시간 - 종료 시간을 시작 시간 기준으로 정렬 
+ 처음 회의실 배정받은 회의의 종료 시간을 우선순위큐에 저장 
+ 이후 회의의 시작 시간 >= 우선순위 큐에 들어간 종료시간이라면 종료 시간을 우선순위 큐에서 뻄 
+ 마지막 우선순위 큐의 크기가 필요한 회의실 개수 
  
## 풀이
```java
import java.io.*; 
import java.util.*;

public class Main{
    public static void main(String[] args) throws IOException{
        
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        int N = Integer.parseInt(br.readLine());

        int[][] meetings = new int[N][2];
        for(int i = 0; i < N; i++){
            StringTokenizer st = new StringTokenizer(br.readLine());
            meetings[i][0] = Integer.parseInt(st.nextToken()); // start
            meetings[i][1] = Integer.parseInt(st.nextToken()); // end
        }
        
        Arrays.sort(meetings, (a,b) -> a[0]-b[0]); //회의 시작 시간 기준으로 정렬
        
        PriorityQueue<Integer> room = new PriorityQueue<>(); 
        room.add(meetings[0][1]);
        
        for(int i =1; i < N; i++){
            if(meetings[i][0] >= room.peek()){
                room.poll();
            }
            room.add(meetings[i][1]);
        }
        
        System.out.println(room.size());
        
    }
}
```
