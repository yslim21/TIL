# 기타 레슨

[기타 레슨](https://www.acmicpc.net/problem/2343)

## 기타 레슨
+ 블루레이 최소 크기는 레슨 중 가장 긴 수 이고 최대크기는 모든 레슨 길이를 합친 수
+ 이진 탐색을 이용해서
  + 최소크기 ~ 최대크기 사이의 중앙값을 구하자
  + 만약 해당 중앙값으로 동영상을 M개에 나눠서 저장할 수 있으면 왼쪽 데이터셋
  + 저장할 수 없으면 오른쪽 데이터셋 탐색 
  + 탐색 종료 후 start값이 블루레이 크기의 최소값 

## 풀이1 - 오답
```java
import java.io.*;
import java.util.*;

class Main{
    static int N,M;
    static int[] arr;
    
    public static void main(String[] args)throws IOException{
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        StringTokenizer st = new StringTokenizer(br.readLine());
        
        N = Integer.parseInt(st.nextToken());
        M = Integer.parseInt(st.nextToken());
        
        arr = new int[N];
        st = new StringTokenizer(br.readLine());
        int end = 0; // 끝 인덱스
        for(int i=0; i<N;i++){
            arr[i] = Integer.parseInt(st.nextToken());
            end += arr[i]; 
        }
        
        int start = arr[N-1]; //시작 인덱스
        
        while(start <= end){
            int mid = (start+end)/2;
            int sum=0;
            int count=0; 
            for(int i=0; i <N; i++){
                if(sum +arr[i]> mid){
                    count++;
                    sum=0; 
                }
                sum += arr[i];
            }
            
            if(count <= M){
                end = mid-1;
            }else{
                start = mid+1;
            }
        }
        
        System.out.println(start);
        

           
    }
}
```

## 풀이2 - 정답
+ 탐색 후 sum이 0이 아니면 블루레이가 1개 더 필요하다는 것을 의미함
+ 강의 길이가 오름차순으로 제시되는것이 아니므로 블루레이 최소길이는 Math.max이용해서 찾아야함 
```java
import java.io.*;
import java.util.*;

class Main{
    static int N,M;
    static int[] arr;
    
    public static void main(String[] args)throws IOException{
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        StringTokenizer st = new StringTokenizer(br.readLine());
        
        N = Integer.parseInt(st.nextToken());
        M = Integer.parseInt(st.nextToken());
        
        arr = new int[N];
        st = new StringTokenizer(br.readLine());
        int start =0;
        int end = 0; // 끝 인덱스
        for(int i=0; i<N;i++){
            arr[i] = Integer.parseInt(st.nextToken());
            start = Math.max(arr[i],start);
            end += arr[i]; 
        }
        
        while(start <= end){
            int mid = (start+end)/2;
            int sum=0;
            int count=0; 
            for(int i=0; i <N; i++){
                if(sum +arr[i]> mid){
                    count++;
                    sum=0; 
                }
                sum += arr[i];
            }
            
            if(sum!=0){
                count++;
            }
            
            if(count <= M){
                end = mid-1;
            }else{
                start = mid+1;
            }
        }
        
        System.out.println(start);
        

           
    }
}
```
