### 날짜 형식 지정
---
```SQL
-- 2025-09-19 (YYYY-MM-DD)
DATE_FORMAT(NOW(), '%Y-%m-%d');

-- 2025/09/19 (YYYY/MM/DD)
SELECT DATE_FORMAT(NOW(), '%Y/%m/%d');

-- 2025-09-19 15:45 (YYYY-MM-DD HH:MM)
SELECT DATE_FORMAT(NOW(), '%Y-%m-%d %H:%i');

-- Sep 19, 2025 (Month Day, Year)
SELECT DATE_FORMAT(NOW(), '%b %d, %Y');

```
